# AI工具测评 · 高效视频操作手册
> 内部学习资料 · 今日起实操 · 2026-05-17

---

## 一、为什么现在做这个赛道

- 门槛低：不用会代码、不用露脸、甚至不用自己配音
- 正反馈快：选题对了第一条就能几万播放，变现比上班快
- 商单稳定：AI工具厂商需求旺盛，几千粉就有人找，保守估计2-3年持续红利
- 账号资产：长期复利，越做越值钱

**三阶段路径：**
- 第一阶段：AI工具/网站测评起号
- 第二阶段：细分赛道深耕（电商+AI、办公提效+AI、副业+AI…）
- 第三阶段：针对用户痛点自建站点，广告+订阅变现

---

## 二、全流程SOP（每条视频必走）

```
确定人群 → 找选题 → 写文案 → 配音 → 素材+剪辑 → 发布
```

### Step 1 · 确定人群（做一次，长期用）

用三类标签锁定你的目标用户：

| 标签类型 | 举例 |
|---------|------|
| 职业标签 | 产品经理、设计师、程序员、教师、自由职业者 |
| 兴趣标签 | 健身、摄影、音乐、阅读、旅行 |
| 身份标签 | 宝爸宝妈、学生、创业者、职场新人 |

**原则：** 选定一个人群后，所有选题都围绕这个人群展开，不要飘。

---

### Step 2 · 找选题（两条路）

**路径A · 对标爆款：**
1. 找同人群的对标账号
2. 看他们的爆款视频选题
3. 只借鉴选题方向，不抄内容
4. 自己去体验工具/网站，写原创文案

**路径B · 痛点驱动：**
1. 想清楚目标人群的真实痛点是什么
2. 根据痛点找对应工具或网站
3. 在平台搜一下验证这个选题有没有流量
4. 跨界迁移：其他赛道的爆款选题可以复用

⚠️ **关键认知：前期先分享「有用的网站」拿流量，AI工具是后期变现手段，不要一开始就推AI工具。**

---

### Step 3 · 写文案（最重要，占工作量70-90%）

**五段式爆款脚本结构：**

```
① 总介绍（5-10秒）
   → 一句大白话讲清能干嘛
   → 三种开头：痛点型 / 利益型 / 拉踩型

② 分介绍
   → 具体功能分点讲
   → 按功能模块展开

③ 递进介绍
   → 挖更深的亮点
   → 说出超出预期的价值

④ 举例介绍
   → 真实场景代入
   → 切中痛点的具体案例

⑤ 结尾号召
   → 免费/低门槛 + 引导评论
   → 不要出现「关注我」「快去试试」等导流话术
```

**实战范例一（人体解剖网站）：**
> 总：这是一个能让你通过解剖视角看清自己器官的网站
> 分：收录了几千种疾病的3D动态模型和手术全过程，医学生/健身教练/普通人都能用
> 递进：更变态的是，人体构造可以360度无死角旋转查看
> 举例：比如女朋友偏头痛，医生说的原理听不懂，搜一下立刻看3D动画演示秒懂
> 号召：网站完全免费

**实战范例二（英语学习网站）：**
> 总：这是一个能帮你省下几万块补课费的神级网站
> 分：一分钱不花，学完新概念英语第一册到第四册全部内容
> 递进：把枯燥课文做成动画片，一边「环游美国」一边学英语，单词/发音/练习题全配齐
> 举例：觉得语法太难？点开视频教学，免费看名师详细讲解，外面按小时收费的课
> 号召：把补习班的钱省下来，它不香吗？

---

### Step 4 · 配音

1. 网上找有抑扬顿挫、好听的参考音色
2. 用AI提炼音色特征描述词
3. 用MiniMax等工具生成配音
4. 让多人试听评价（不能只靠自己判断）

---

### Step 5 · 素材+剪辑（轻剪辑）

- 每句文案对应一个网站画面（写文案时就要规划录屏）
- 不插入第三方无关素材
- 声音 + 画面 + 文案 三维一致
- 轻剪辑：录屏 + 配音即可，不需要花哨特效

---

### Step 6 · 发布（一鱼多吃）

**全平台同步：** 抖音 / 小红书 / B站 / 快手 / 视频号

好处：
- 流量互补：A平台没火，B平台可能爆
- 减缓焦虑：不依赖单一平台数据
- 边际成本低：视频做好了，多发一个平台只需1分钟

## 三、平台推荐机制

```
500基础播放 → 数据达标 → 进入下一层 → 继续达标 → 持续推荐
```

**核心考核指标（按重要性排）：**
- 点赞：内容有用就会点
- 收藏：想用的才会收藏
- 评论：有代入感才会评论

**转化路径：**
```
用户评论想要 → 回复粉丝群链接 → 设置关注才能入群 → 完成关注转化 → 群内持续运营
```

## 四、变现路径

| 阶段 | 方式 | 门槛 |
|------|------|------|
| 早期 | 平台创作激励（抖音伙伴计划等） | 粉丝量达标 |
| 中期 | 品牌广告合作（AI厂商主动来找） | 几千粉即可 |
| 中期 | 海外渠道（内容搬运+海外AI工具推广） | 翻译+发布 |
| 后期 | 自建工具站（订阅/按次收费+广告） | 有用户基础 |
| 后期 | 知识付费/带徒 | 有经验积累 |

## 五、避坑清单

| ❌ 不要做 | ✅ 要做 |
|---------|--------|
| 视频里出现网址/二维码/QQ号 | 在评论区引导私信 |
| 说「快去试试」「关注我」 | 说「完全免费」「评论区告诉我」 |
| 一比一复刻别人视频 | 借鉴选题，原创文案 |
| 让AI全自动写文案不修改 | AI出初稿，自己改出人味 |
| 插入无关搞笑/擦边素材 | 画面严格对应文案 |
| 外包剪辑给不懂行的人 | 前期自己动手，轻剪辑门槛低 |
| 一开始就推AI工具 | 先分享有用网站积累粉丝 |
