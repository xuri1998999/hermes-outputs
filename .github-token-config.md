# GitHub Token 配置说明

海拓的 GitHub PAT 不得直接写入仓库文件。需要通过环境变量配置。

## 广州服务器配置

在 `haituo.js` 或 docker-compose 环境变量中设置：

```bash
GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## 本地开发配置

```bash
export GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## GitHub Secrets（推荐用于 CI/CD）

在 GitHub 仓库 Settings → Secrets → Actions 中添加：

- Name: `GITHUB_TOKEN`
- Value: 你的 Personal Access Token

然后在 Actions workflow 中用 `${{ secrets.GITHUB_TOKEN }}` 引用。
