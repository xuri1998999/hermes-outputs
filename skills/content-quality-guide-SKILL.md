# 笔耕创作质量审核规范 v1.1

> 本规范注入笔耕创作流程，所有创作产出必须通过自检后方可发布

---

## 一、发布前自检清单（6项全过才发）

- [ ] **标题格式正确**：无"选题："前缀，纯标题文本，可直接用作公众号/小红书标题
- [ ] **摘要格式正确**：纯文本，无HTML标签，无JSON结构，长度50-200字
- [ ] **封面图已生成**：公众号/小红书封面图已生成且上传飞书获取永久URL
- [ ] **内容长度合规**：公众号800-2000字，小红书300-800字
- [ ] **人称规范**：开头有"我"的真实场景，结尾有可直接用的模板
- [ ] **结尾行动钩子**：结尾有"拿去用"/"评论区见"/"回复XX拿模板"等行动，非空祝福

---

## 二、人称规范（🔴 铁律，2026-04-24新增）

### 核心要求：每篇文章必须有"我"
- **开头**：必须用"我"的真实经历/场景开头，禁止空谈
  - ✅ `上周帮同事用AI干了X，他以为我加班到半夜`
  - ✅ `我用X方法省了Y时间/钱了`
  - ❌ `在AI时代，如何提升效率...`（空泛）
  - ❌ `随着人工智能的快速发展...`（AI味开头）
- **结尾**：必须有读者可直接带走的模板/工具/步骤，禁止空祝福
  - ✅ `我的AI工作流模板，可以直接用（附链接）`
  - ✅ `评论区留言'模板'，我把prompt发给你`
  - ❌ `希望对你有帮助`（空泛）
  - ❌ `以上就是今天的分享，感谢阅读`（无价值）
- **全文**：始终以第一人称叙述，穿插"我的判断"、"我的踩坑"、"我的实测"

### 人称检查项
- [ ] 开头有"我"的具体场景（非泛泛而谈）
- [ ] 中间有"我"的判断/实测/踩坑细节
- [ ] 结尾有读者可直接用的模板/工具/步骤

---

## 四、正文写作规律（🔴 铁律，2026-04-24新增）

> 来源：苍何+饼干哥哥内容模式提炼

### P0 铁律（必须遵守）
1. **开头必须让人停下来**：不废话，直接场景
   - ✅ `卧槽`、`摸鱼`、`你是不是也...`、`上周帮同事用AI干了X`
   - ❌ `随着AI的快速发展...`、`在当今时代...`
2. **技术内容但要有人味**：代码/工具可以多，但要穿插"我的判断"和"踩的坑"
3. **结尾给读者一个行动**：不是空祝福，而是"拿去用"/"评论区见"/"回复XX拿模板"/"收藏下周继续"
4. **有图有真相**：能描述截图就说清楚（因为实际生成不了），让读者能想象画面

### P1 加分项
5. **先说Why再做How**：先说这件事为什么值得解决，再说用什么工具
6. **承认踩过的坑**：穿插"我第一次试的时候翻车了"、"npm卡顿了我折腾了2小时"——细节让人感觉真实
7. **追更钩子**：结尾统一加"关注我，X天后继续这个系列"/"评论区聊聊你遇到的情况"

### 人称+结尾双检项
- [ ] 开头有"我"的具体场景（非泛泛而谈）
- [ ] 中间有"我"的判断/实测/踩坑细节
- [ ] 结尾有读者可直接用的模板/工具/步骤
- [ ] 结尾有行动钩子（非空祝福）

## 五、标题规范

### 合格标题特征
- 有信息量（包含具体数字/工具名/事件）
- 有情绪钩子（反常识/痛点/好奇）
- 可直接用作公众号/小红书标题，无需二次编辑

### 不合格标题示例
```
❌ "选题：Anthropic封禁OpenClaw创始人"   ← 带"选题："前缀
❌ "AI工具推荐"                            ← 太泛，无信息量
❌ "好消息！xxx来了"                       ← 空泛感叹
❌ "【重磅】xxx"                          ← 标题党
```

### 标题5种公式
1. **数字+痛点**：`5个AI工具，让我每天多出2小时`
2. **反常识**：`不用Midjourney，我用这个免费工具做出更好看的图`
3. **对比**：`Claude vs GPT-4：程序员实测1个月，结论出乎意料`
4. **教程**：`如何用AI在10分钟内完成一周的工作报告`
5. **踩坑**：`我用AI写周报被老板看出来了，这3个坑别踩`

## 三、正文规范（⚠️ 必须加Emoji）

> **2026-04-20 更新**：所有平台输出必须带Emoji，这是发布标准，不是可选项

### 公众号文章
- 开头：200字内抛出痛点/反常识/数据
- 正文：分段清晰，每段不超过200字，段首可加 📌🔖📊💡 等emoji
- 结尾：有Call-to-Action（关注/收藏/评论区见），加 🎯👇
- 禁止：AI味开头（"随着人工智能的快速发展..."）

### 小红书笔记（emoji是核心要求）
- 开头：3行内抛出价值，段首必须带emoji如 💡🔥⚠️✅
- 正文：每段都要emoji点缀，常用emoji表：
  - 重点强调：`🔴⚠️🔥💡⭐`
  - 步骤列表：`① ② ③` 或 `✅`
  - 情绪表达：`绝了👍太牛了😭笑死😂`
  - 话题标签：`#️⃣`
  - 互动引导：`👇评论区见🙋‍♀️求关注❤️`
- 标签：3-5个相关话题标签
- 结尾：互动引导必须带emoji（`👇评论区聊聊`）
- **小红书缺少emoji = 内容不合格，打回重写**

## 四、封面图规范

### Seedream生图Prompt模板
```
杂志封面风格，[核心画面描述]，科技感，[情绪氛围]，
精致光影，居中构图，竖版9:16，[背景描述]，[画面风格]
```

### 合格封面图标准
- 图片尺寸：公众号900x383，小红书1440x2560
- 画面清晰主体突出
- 文字标题可直接覆盖在图片上

## 五、审稿红线（触犯即打回重写）

1. 标题无信息量或无实操价值
2. 正文开头AI味明显（套路开场白）
3. 全文只说好不说坏，无客观对比
4. 内容与素材来源严重不符
5. 标题包含派活原文（如"选题：xxx"）
6. 摘要含HTML标签或JSON结构
7. 缺少封面图或封面图与内容不符
8. **小红书/公众号正文缺少emoji** ← 2026-04-20新增

## 六、发布流程

### 公众号
1. 自检4项全过
2. 生成封面图 → 上传飞书 → 获取永久URL
3. 发微信草稿箱
4. 入多维表格「内容创作表」
5. 入多维表格「小红书内容库」封面图URL

### 小红书
1. 自检4项全过
2. 生成封面图 → 上传飞书 → 获取永久URL
3. 写5张卡片图
4. 入多维表格「小红书内容库」
5. 发布小红书
