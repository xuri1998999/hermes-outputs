# 海拓输出规范

## 核心原则

- **每条输出必须包含所有成品字段**，不能只输出摘要
- **自检在前，分类在后**，自检未通过不得标注✅
- **不输出用户截图描述**，只引用文字内容
- **有争议/法律风险的内容必须标注⚠️**

---

## 成品标准格式（每条必填）

每条AI工具速测输出必须包含以下8个字段，按顺序输出：

```
【AI工具速测·今日精选】
工具：[工具名称]
争议：[一句话争议点，无则写"暂无明显争议"]
对比：[与竞品的关键数据对比，含价格/性能/速度差异]
话题：[引发讨论的行业事件或趋势]
抖灵标题：[28字以内，含[PH]平台标记+emoji]
小红书标题：[26字以内，含数字或情绪词]
封面提示词：[Seedream专用，1440×2560，9:16竖版，无文字，描述画面场景]
脚本开头：[抖灵3秒钩子，分行写，含音效/转场标注]
竞品数据：[价格+核心能力对比，用表格或列表]
✅自检全过 / ⚠️自检未过：[说明哪项未通过]
```

---

## 自检清单（每条输出前必查）

| 检查项 | ✅通过条件 | ❌未通过 |
|--------|-----------|---------|
| 工具名称 | 有官方名称，附来源链接 | 只有昵称/代号 |
| 争议点 | 有具体事件，非空泛描述 | "争议很大"等模糊表达 |
| 竞品对比 | 有具体数据（价格/速度/准确率等） | "比竞品好"无数据 |
| 抖灵标题 | ≤28字，含[PH]/[HN]/[Reddit]标记 | 无平台标记或超28字 |
| 小红书标题 | ≤26字，含数字/情绪词/工具名 | 纯描述无爆点 |
| 封面提示词 | 含场景+风格+尺寸，无具体公司Logo | 含Logo或无尺寸 |
| 脚本开头 | 3秒内，含音效/黑屏/字幕标注 | 无音效标注或太长 |
| 用户评价 | 引用文字，不描述截图 | 出现"头像马赛克"等截图描述 |
| 金额数据 | 有来源或标注"待核实" | 无来源且未标注 |

## 分类判定规则

### ✅ 直接入库（满足全部条件）
- 自检8项全部通过
- 无用户截图描述
- 金额/数据有来源依据
- 来源稳定（ProductHunt/Futurepedia/HN/Reddit）
- 无法律/监管敏感内容

→ 动作：直接写入信息池，标注`✅可入库`，发海媒总群通知Aily

### ⚠️ 需Aily审核（满足任一条件）
- 涉及用户截图描述（即使打了马赛克）
- 金额/数据无法核实（无来源）
- 涉及负面/法律/监管事件（封禁/起诉/诈骗）
- 新分类（不在预设分类库）
- 数据矛盾（多处来源数据不一致）

→ 动作：标注`⚠️需Aily审核`，说明具体原因，发海媒总群通知Aily处理

### 🔴 暂不入库（满足任一条件）
- 来源不稳定（匿名来源/自媒体）
- 事实性错误（明显数据造假）
- 数据矛盾且无法核实
- 内容与海媒定位不符

→ 动作：标注`🔴暂不入库`，说明原因，打回海拓补充或重查
